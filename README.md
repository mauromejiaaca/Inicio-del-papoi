# 🎰 Mini Lotería - Casi imposible de ganar

Simulador de lotería en Python donde ingresas 6 números
y rezas para que coincidan con los 6 aleatorios del programa.
Spoiler: no van a coincidir.

## ¿Cómo se juega?

1. Ejecuta el programa
2. Ingresa 6 números del 1 al 48
3. Observa cómo la lotería te ignora completamente

## Requisitos

- Python 3.x
- Esperanza (opcional)

## Ejecutar

python Loteria.py

## Lo que aprendí haciéndolo

- Bucles while
- Listas y cómo evitar duplicados
- Que la lotería es una estafa
